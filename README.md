# Сайт поиска компании по своместным увлечениям


fronend составляющая сайта

  - Страница регистранции и авторизации
  - Главная страница
  - Страница профиля пользователя для добавления и редактирования его интересов
  - Страница с предложениями других участников проекта наиболее совместимых по жизненным приоритетам

## Установка

  - Скачать проект и распаковать в папку на диске
  - Установить [DOCKER](https://www.docker.com/) согласно инструкции на сайте
  - Установить [Python](https://www.python.org/) согласно инструкции на сайте

## Начальная настройка и запуск:
#### Настройка проекта
  - отредактировать файл .env в корне проекта в соответствии с необходимыми настройками базы данных и подключения к электронной почте
  - При необходимости изменения порта сайта отредактировать файл docker-compose.yml в секции 
```  
web:
    port:
        -8000:800
```
#### Запуск проекта
 - открыть терминал в папке проекта
 - произвести запуск проекта
 ``` 
 docker-compose up
 ```
 - Если необходимо запустить в фоновом режиме можно добавить к команде запуска ключ -d
 - Для остановки контейнеров запущенных в фоновом режиме можно воспользоваться коммандой
 ```
 docker-compose down
 ```
 - Для тестирования проекта, можно добавить в него фейковых пользователей, для этого нужно перейти в папку проекта и запустить скрипт
 ```sh
 cd project
 python add_fake.py
 ```
>Работа по добавлению фейковых пользователей производится в фоновом режиме. До добавления всех пользователей может наблюдаться задержка отправки электронной почты пользователям
## Работа с сайтом

После запуска проекта можно его протестировать зайдя по адресу [http://localhost:8000](http://localhost:8000) (Если не был изменен порт в настройках докер контейнера). Откроется страница авторизации и регистрации. Т.к. учетки у нас нет прийдется зарегестрироваться. При регистрации на электронную почту отправится письмо со ссылкой на подтверждение, при переходе по которой учетная запись будет подтверждена и можно авторизироваться. При авторизации аналогичная процедура с отправкой одноразового кода для подтверждения авторизации.

После успешной авторизации на сайте можно будет приступить к добавлению собственных приоритетов на вкладке профиля или для просмотра рейтинга совместимости на вкладке рекомендации
